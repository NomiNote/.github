*Environmental Monitoring System*

<p align="center">
  <img src="https://img.shields.io/badge/WebSocket-black?style=for-the-badge&logo=socket.io&badgeColor=010101" alt="WebSocket"/>

> - ⚡ WebSocket-based real-time data streaming
